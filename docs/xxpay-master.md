# 一、项目概览

## 1.1 简介
- XxPay聚合支付 XxPay聚合支付使用Java开发，包括spring-cloud、dubbo、spring-boot三个架构版本，已接入微信、支付宝等主流支付渠道，可直接用于生产环境

- 目前已经接入支付渠道：微信(公众号支付、扫码支付、APP支付、H5支付)、支付宝(电脑网站支付、手机网站支付、APP支付、当面付)

## 1.2 环境
```aidl
JDK1.8、SpringBoot、SpringCloud、Dubbo
```
